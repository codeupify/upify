# Commands

## init
Initialize a new project.

```bash
upify init
```

## platform add
Add platform support to your project.

```bash
upify platform add aws
upify platform add gcp
```

## deploy
Deploy your application to the specified platform.

```bash
upify deploy aws
upify deploy gcp
```

## Flags

### Global
- `--help`: Display help information
